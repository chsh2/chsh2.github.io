    概述 #

    NijiGPen是免费开源的Blender插件，开发的目的是为Grease Pencil提供更多2D绘画与平面设计方面的功能，包括：

    • 2D矢量多边形算法
    • 快速线稿清理与铺色
    • 由2D线条生成3D物体
    • 用于其它绘画/设计软件的导入导出与格式转换

    本插件的名称来源于“二次元”（nijigen）与Grease Pencil简写（gpencil）的组合。

    GitHub YouTube Gumroad

    当前版本：0.12.0

    （本工具的部分功能依赖于多种优化算法，但它们均不需要外部模型或训练样本。本工具没有使用他人的艺术作品作为算法的训练材料。）

    致谢 #

    插件使用了下列软件包来实现功能:

    • Pyclipper wrapper by Maxime Chalton and the Clipper library by Angus Johnson
    • Triangle by Dzhelil Rufat and Jonathan Richard Shewchuk
    • SciPy
    • Scikit-image

    另外一些算法的实现受到多篇学术论文的启发，在算法设计页面有完整的参考资料列表。
